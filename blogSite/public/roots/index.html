Roots

This is one of the earliest programming exercises I had worked on: Given 3 parameters, a function would return how many roots an equation had and what they were.

Sample Code:

def quadratic_roots(a, b, c):
    """
    This function will take the discriminant and use it to see how many
    roots exist for the equation. It then finds what the roots are. The a, b, c variables pertain to the
    a,b,c in the equation ax^2 + bx + c.
    """
    discrim = (b**2 - 4 * (a * c))
    print("Equation: ", a,"x^2 +", b, "x +", c, sep="")

    if discrim > 0:
        print("There's 2 roots")    

        root1 = (-b + math.sqrt(discrim)) / (2*a)
        root2 = (-b - math.sqrt(discrim)) / (2*a)
        print("One of the roots are: ", root1)
        print("The other root is: ", root2)
        
    elif discrim == 0:
        print("There's 1 root")
        
        root = (-b + math.sqrt(discrim)) / (2*a)
        print("The root is: ", root)
    elif discrim < 0:
        print("There's 0 roots")
